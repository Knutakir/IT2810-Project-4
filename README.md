# Project 4

## Install and Run

The client is set up to use the server on the VM, to use your own server with database ...

### Server
```sh
$ cd server
$ npm install
$ npm run init-db
$ npm start
```

### Client
```sh
$ cd client
$ npm install
$ npm start
```
